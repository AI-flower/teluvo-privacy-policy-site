What to Include in Your Request

  • App version (for example, 1.0)
  • Device model and iOS version
  • A brief problem description and steps to reproduce
  • Optional screenshots or error messages

Troubleshooting Basics

  • Check network stability and retry the action.
  • Update Teluvo to the latest version.
  • Re-open the app and re-run the failed task.
  • For server connection issues, verify host, port, and credentials.
